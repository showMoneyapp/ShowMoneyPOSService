# ShowMoneyPOSService

### Configure your 'conf/config.ini' before running

  # RPC node：
  RPC_url = "http://localhost:8332"
  # RPC userName：
  RPC_username = "root"
  # RPC password：
  RPC_password = "password"

  # POS-API-Service Port：
  POS_API_Port = 2323
  # POS-WS-Service Port：
  POS_WS_Port = 1212
